What is Internet ?

The internet is a global network that connects computers and devices,

enabling data sharing and communication using standardized protocols like TCP/IP.

It powers services like the web, email, and streaming, operating without a central authority.

What is TCP/IP ?

TCP/IP is a set of protocols that enables devices to communicate over the internet.

TCP ensures reliable data transfer, while IP handles addressing and routing to get data to the right destination.

TCP/IP is like a postal system for the internet

IP is the address on the envelope, making sure the package (data)

to the right house (devices)

TCP is the delivery service that ensure the package arrives safely

, complete, and in the right order.
